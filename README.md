这个不只是为了做聊天室而做的，通过socket通信可以完成很多功能。这就是我要实现的东西，简单，实用。<br />

功能其实很简单，希望能通过简单的功能做出好用的东西，因为做的比较匆忙，有些BUG、功能也没有时间去修复和完善，如果可以的话，可以大家一起来完善，成为一个真正可以拿来用的项目。<br />
